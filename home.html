About the Project

This platform is developed as a joint project for CPE210 Database Systems and CPE212 Internet Based Programming courses. The system features secure user authentication, a dynamic quiz engine, and a competitive leaderboard. Our goal is to demonstrate both client-side interactive UI and robust server-side data management.
