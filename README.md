# 🚀 The Oneionner - Online Quiz Platform

Karabük Üniversitesi **CPE210 Database Systems** ve **CPE212 Internet Based Programming** dersleri için geliştirilmiş ortak bir projedir.

## 📝 Proje Hakkında
Öğrencilerin bilgilerini test edebileceği, rekabetçi bir çevrimiçi sınav platformudur. Sistem; kullanıcı kimlik doğrulaması, dinamik sınav motoru ve anlık güncellenen bir liderlik tablosu sunar.

## 🛠️ Kullanılan Teknolojiler
* **Backend:** PHP (PDO ile güvenli veritabanı bağlantısı)
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Veritabanı:** MySQL
* **İletişim:** AJAX (Fetch API ile sayfa yenilenmeden veri transferi)
* **Güvenlik:** SHA-256 (Password Hashing)

## 🎯 Özellikler
* **Güvenli Giriş:** PDO kullanarak SQL Injection korumalı login sistemi.
* **Akademisyen Paneli:** Soru havuzunu yönetme, ekleme ve düzenleme.
* **Dinamik Sınav:** Seçilen kategoriye göre anlık yüklenen sorular ve zamanlayıcı.
* **Liderlik Tablosu:** Öğrencilerin rekabet edebileceği güncel sıralama sistemi.
